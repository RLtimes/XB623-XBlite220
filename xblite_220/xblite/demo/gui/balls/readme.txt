Created using XBLite

dszafranski@wanadoo.fr

To run screensaver program, just
double click on .src file.

To install, right click on file
and select install or manually
copy colliding balls.scr
to the \windows\system directory.

Right click on main desktop window
to get popup context menu, select
Properties to get Display Properties
dialog box. Click on Screen Saver tab.

Then in screen saver drop listbox, select
Colliding Balls. Press Settings button
to modify settings, or press preview to
run screensaver.
